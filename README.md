# Giới thiệu dự án
Dự án xây dựng một hệ thống bán hàng trực tuyến (E-commerce Web App) nhằm cung cấp nền tảng mua sắm tiện lợi, hiện đại cho người dùng, đồng thời hỗ trợ doanh nghiệp quản lý hoạt động kinh doanh hiệu quả.
Hệ thống bao gồm hai phân hệ chính là Khách hàng và Quản trị, cùng với các tính năng nâng cao ứng dụng công nghệ AI.

# Mô tả dự án
1. Phân hệ Khách hàng
- Khám phá: tìm kiếm, lọc, xem sản phẩm 
- Tương tác: đánh giá, bình luận, wishlist 
- Giao dịch: giỏ hàng, mã giảm giá, đặt hàng, thanh toán 
- Cá nhân: quản lý tài khoản, địa chỉ, lịch sử mua hàng 
- Chatbox AI: 
  + Tư vấn sản phẩm 
  + Trả lời câu hỏi tự động 
  + Hỗ trợ tra cứu đơn hàng 
- Theo dõi đơn hàng: 
  + Trạng thái:
  + Chờ xác nhận → Đã xác nhận → Đang đóng gói → Đang giao → Hoàn thành / Hủy 
  + Hiển thị timeline + thông báo khi thay đổi 
2. Phân hệ Quản trị
- Quản lý sản phẩm & kho 
- Quản lý đơn hàng (cập nhật trạng thái, xử lý giao hàng) 
- Quản lý người dùng & phân quyền 
- Báo cáo & thống kê doanh thu 
- Quản lý Chatbox AI 
- Quản lý trạng thái đơn hàng (real-time) 
 3. Tính năng nâng cao
- Theo dõi đơn hàng chi tiết
- Tích hợp thanh toán online (mở rộng) 
- Gợi ý sản phẩm theo hành vi người dùng
- Bảo mật nâng cao
